# album-genre-predictor
Utilizing transfer learning on DenseNet121 to predict musical genre based on album cover art

The dataset and description can be found here: https://www.kaggle.com/datasets/jaidityachopra/esg-sustainability-reports-of-s-and-p-500-companies

The Medium article detailing our data, methods, analysis, and findings can be found here: https://medium.com/@ecantero/predicting-album-genres-0b6d143a789c
